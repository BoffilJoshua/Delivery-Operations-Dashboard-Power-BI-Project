## Project Overview
This project analyzes logistics delivery performance using Power BI. The dashboard provides insights into operational efficiency, delivery reliability, cost trends, and customer satisfaction.

## Key KPIs
- Total Deliveries
- On-Time Delivery %
- Total Revenue
- Average Delivery Time
- Customer Rating

## Features
- Delivery trend analysis
- Weather impact on delays
- Cost vs distance efficiency
- Partner performance comparison
- Running revenue totals

## Tools Used
- Power BI
- DAX
- Data modeling

## Outcome
The dashboard helps identify delay patterns, cost drivers, and performance gaps, supporting better operational decisions.
